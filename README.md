# 🚀 Gradle Java Project
This project demonstrates how to build and run a Java application using Gradle.

## 🛠️ Tools & Technologies
- Java 8
- Gradle
## 🔄 CI/CD Concept
Gradle is used in CI/CD pipelines to automate:
- Build process
- Testing
- Deployment

It helps in continuous integration by automatically building the project whenever code changes.
